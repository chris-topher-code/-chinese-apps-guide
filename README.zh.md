# Chinese Apps Guide 中国应用指南

这是一个重新设计的静态网站，用来帮助外国人了解在中国日常生活中常用的重要应用。

原项目参考：

https://github.com/chris-topher-code/-chinese-apps-guide

## 这个版本做了什么

这个版本保留了原项目「中国应用目录」的想法，但把体验改成更实用的「来华第一周应用指南」，面向游客、留学生、商务访客和新居民。

功能包括：

- 移动端优先的旅行指南风格界面。
- 纸张质感、红色印章元素和更有编辑感的字体风格。
- 英文/法文切换。
- 入门套装：到达、支付、交通、吃喝、购物、学生生活和长期居住。
- 支持按应用名、中文名、任务、分类或设置提示搜索。
- 分类筛选。
- 保留上游原项目的完整应用覆盖，并新增 Dianping。
- 对容易重复的概念做合并：WeChat Pay 放进 WeChat 说明里，不再作为第二张 WeChat 卡片展示。
- 应用卡片使用真实远程应用图标，并保留本地首字 fallback。
- 详情弹窗展示设置步骤、注意事项、类似西方应用和使用场景。
- 获取应用时提示选择直接 iOS App Store 页面、安卓应用页面/搜索或网页搜索。
- 页面、卡片和入门套装有轻量动画，并支持 `prefers-reduced-motion`。
- 无需构建步骤，无依赖。

## 如何运行

直接用浏览器打开 `index.html`。

也可以启动本地服务器：

```bash
python3 -m http.server 8080
```

然后打开：

```text
http://localhost:8080
```

## 设计方向

这个界面希望更像一本实用的中国生活指南，而不是普通的应用列表。

核心思路：

- 优先考虑第一周是否有用，而不是只看应用是否热门。
- 提前展示设置难度和外国人可能遇到的限制。
- 按真实需求组织内容，例如支付、出行、外卖和购物。
- 保留原项目的双语指南思路，同时让视觉风格更有辨识度。

## 应用商店链接

「获取应用」操作会让用户选择平台：

- 如果找到公开匹配项，则打开对应的 iOS App Store 页面。
- 安卓应用页面或搜索。
- 网页搜索备用。

安卓和网页操作刻意不提供原始 APK 文件下载。对公开指南来说，应用商店页面和官方搜索结果更安全，因为应用上架状态、地区限制和下载规则都可能变化。

## 应用图标

应用卡片会在可用时引用公开的 App Store 图标地址。如果远程图标加载失败，界面会自动回退到本地首字图标，保证布局稳定。

## 与原项目兼容

应用列表保留了原项目的全部应用，而不是缩小范围。同时新增 Dianping，将 WeChat 同时归入社交和支付相关场景，把 WeChat Pay 合并到 WeChat 中避免重复卡片，并把 携程 标注为更符合中国语境的 Ctrip。
